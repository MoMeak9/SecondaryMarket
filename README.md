# 校园二手商城（前端）
## 项目简介
西二第五轮考核作业

一个功能基本完全的校园二手交易网站，已完成并部署所有既定需求，用户交互等还有待改进，继续努力！！
## 技术栈
Vue全家桶 + Element_Ui
## 项目使用
安装： 'npm install'
热加载： 'npm run serve'
构建项目：'npm run build'
## 其他引用
- VueParticles粒子背景库
- iconfont 图标库
- axios qs 等库
